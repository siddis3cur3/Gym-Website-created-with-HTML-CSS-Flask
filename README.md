# Gym Website (HTML, CSS & Flask)

A simple gym website built using **HTML**, **CSS**, and **Flask** (Python web framework).  
This project demonstrates a basic full-stack web application structure with static front-end pages and Flask as the backend server.

## About

This repository contains the source code for a gym website designed using HTML and CSS, and served with Flask. It can be used to learn how to integrate basic front-end assets with a Python backend using Flask.

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
    ```bash
    git clone https://github.com/siddis3cur3/Gym-Website-created-with-HTML-CSS-Flask.git
    cd Gym-Website-created-with-HTML-CSS-Flask
    ```

2. **Create and activate a Python virtual environment (optional but recommended)**
    ```bash
    python -m venv venv
    source venv/bin/activate   # macOS/Linux
    venv\Scripts\activate      # Windows
    ```

3. **Install dependencies**
    ```bash
    pip install Flask
    ```

4. **Run the application**
    ```bash
    flask run
    ```

5. **Open in browser**  
   Visit `http://127.0.0.1:5000` to view the site.
