# Gym Website using HTML, CSS, and Flask

This project is a simple gym website built using HTML and CSS for the frontend and Flask (Python) for the backend. The purpose of this project is to demonstrate how a basic website can be served using Flask while maintaining a clean separation between frontend and backend components.

It is intended for learning and practice, especially for beginners exploring web development with Python and Flask.

## Purpose

- To practice building static web pages with HTML and CSS
- To understand how Flask serves web pages
- To demonstrate a basic full-stack web application structure

## Running the Project Locally

Follow the steps below to run the project on your local machine.

### Prerequisites

- Python 3 installed
- pip (Python package manager)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/siddis3cur3/Gym-Website-created-with-HTML-CSS-Flask.git
   cd Gym-Website-created-with-HTML-CSS-Flask
2. Install Flask:
   ```bash
   pip install flask
3. Run the Application:
   ```bash
   flask run
4. Open in Browser:
   ```bash
   Open your browser and go to: http://127.0.0.1:5000

Now you have your own website running locally.
